# VoiceEmo

Speech emotion recognition — comparing Wav2Vec2, HuBERT, and WavLM on RAVDESS with cross-corpus evaluation on CREMA-D.

NLP703 (Speech Processing) course project · MBZUAI · April 2026 · Karim Mahfouz.

## Setup

```bash
pip install -r requirements.txt
```

PyTorch is installed separately per [pytorch.org](https://pytorch.org).
